## üìñ How to Use

1. Go to any **YouTube channel page**
2. Refresh the page (important, otherwise the extension cannot find it)
3. Click the **HydraTracker** extension icon
4. Click **Capture Current Channel**
5. Repeat (1-4) for as many channels as you want (all stay until you close the browser)
6. Click **Save as CSV** when you're ready

Your CSV is now ready to import into HydraWatch!

## üõ† Troubleshooting

**Accidentally captured the wrong channel?**

- Nuke the whole list via **Clear List** to start fresh, or
- Export the CSV and simply delete the unwanted row in any spreadsheet app  (Excel, Google Sheets, Notepad, text editor, etc.)
